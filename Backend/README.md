# English

## Description

This application compose the backend of the Food Share: Brasil project. It was made using NestJS, a powerful NodeJS framework for building APIs with the JavaScript or TypeScript languages. The connection with database is being made with TypeOrm, a tool that facilitates the communication between the application and the database. The database itself is being used as a Docker container with the MySQL image. 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## API Documentation

To access the API documentation, run the application and then search for the following URL on your prefered browser:
```
http://localhost:3000/api
```

# Português

## Descrição

Esta aplicação compõe o backend do projeto Food Share: Brasil. Ela foi feita em NestJS, um poderoso framework de NodeJS para a construção de APIs com as linguagens JavaScript ou TypeScript. A conexão com o banco de dados está sendo feita com TypeOrm, uma ferramenta que facilita a comunicaçaõ entre a aplicação e o banco. O banco de dados por si só está sendo utilizado em um contâiner Docker que foi criado com a imagem do MySQL.
